Otto the Shepherd Dog

Developed a 3D herding simulation in Unity featuring a custom flocking AI based on Reynolds' Boid algorithm. Extended the boid algorithm with a fourth “Dog force” for realistic sheep herding behavior. Implemented capsule collision physics, player controls, sound design, and environmental particle effects.

Unity C# Physics

Ahoy.IO

Built a real-time multiplayer battle boat game using THREE.js for 3D graphics and Socket.IO for networking. Designed projectile handling, player sync, and interpolation for smooth gameplay while minimizing bandwidth to the server to reduce hosting costs. Features procedural terrain generated with Perlin noise using multiple octaves to create unique maps for each lobby.

JavaScript Node.js Express Socket.IO THREE.js

Baxter Robot Tic-Tac-Toe

A Tic-Tac-Toe playing robot using Baxter research platform, integrating computer vision (OpenCV), inverse kinematics (MoveIt), and the MiniMax AI algorithm. On our team I was in charge of implementing the MiniMax algorithm with alpha-beta pruning. This reduced the Tic-Tac-Toe decision tree size by 60% while having perfect play against opponents.

Python Ai (Minimax) Robot Operating System
