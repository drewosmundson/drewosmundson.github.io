Otto the Shepherd Dog

Developed a 3D herding simulation in Unity featuring a custom flocking AI based on Craig Reynolds' Boids algorithm. Extended the Boids algorithm with a fourth “Dog force” for realistic sheep herding behavior. Implemented capsule collision physics, player controls, sound design, and environmental particle effects.

Unity C# Physics

Ahoy.io

Built a multiplayer battle boat game using THREE.js for 3D graphics and Socket.IO for networking. Designed projectile handling, player sync, and interpolation for smooth gameplay while minimizing bandwidth to the server to reduce hosting costs. Features procedural terrain generated with Perlin noise using multiple octaves to create unique maps for each lobby.

JavaScript Node.js Express Socket.IO THREE.js

Jeoprody Buzzer

Created a real time quiz buzzer web app where users join rooms, the host controls the timer, and players compete agaist each other to get the fastest time. Implemented low latency communication with Socket.IO and server logic with Node.js and Express. Built for the classroom for Jeoprody like group games and quizzes where speed and order is important.
